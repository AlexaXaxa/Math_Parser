# GeoGebraWPF

A simple graphing calculator built in C# and WPF.

## Features
- Add points and lines
- Add polygons

## Future Features
- Plot `y = x + 1` and other equations
- Zoom and pan with touch

## How to Run
1. Clone this repository
2. Open the `.sln` in Visual Studio
3. Press **Run**
